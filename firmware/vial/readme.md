# Vial Firmware for RP2040

This repository contains the Vial firmware for the Peghammer. This firmware enables real-time keymap configuration via the Vial client for the wired RP2040 microcontroller.

##  How to Build and Flash

  Drag and drop the  `.uf2` file onto the RP2040 board while it is in bootloader mode.
