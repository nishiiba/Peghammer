# Peghammer – Hardware Design (PCB & 3D Parts)

This directory contains the hardware design files for the **Peghammer**
left-hand stick keyboard.

## Contents

- PCB design files (KiCad schematics and board files)
- Manufacturing data (Gerber files, drill files, etc.)
- 3D model files (STL) for physical hardware parts
